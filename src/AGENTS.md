# AGENTS — src (продолжение корневого)

Роутер по коду библиотеки. Домены — отдельные папки; в каждой может быть свой AGENTS.md (глубже по дереву).

**Навигация:** по задаче открой AGENTS.md нужного домена или зайди в подпапку и открой её AGENTS.md.

| Домен | Назначение | AGENTS.md |
|-------|------------|-----------|
| [ai](ai/AGENTS.md) | Модели LLM, ядро, агент | [ai/AGENTS.md](ai/AGENTS.md) |
| [commit-gen](commit-gen/AGENTS.md) | Генерация commit-сообщений | [commit-gen/AGENTS.md](commit-gen/AGENTS.md) |
| [cat-gen](cat-gen/AGENTS.md) | Вывод файлов (XML/MD) | [cat-gen/AGENTS.md](cat-gen/AGENTS.md) |
| [review-gen](review-gen/AGENTS.md) | Генерация ревью, specs, фикстуры | [review-gen/AGENTS.md](review-gen/AGENTS.md) |
| [review-verifier](review-verifier/AGENTS.md) | Только архитектура (README); код команды в cli | [review-verifier/AGENTS.md](review-verifier/AGENTS.md) |
| [git](git/AGENTS.md) | Diff, core (exec, files) | [git/AGENTS.md](git/AGENTS.md) |
| [vcs-client](vcs-client/AGENTS.md) | GitLab: MR, discussions, клиент | [vcs-client/AGENTS.md](vcs-client/AGENTS.md) |
| [prompts](prompts/AGENTS.md) | Шаблоны промптов (commit, review, emoji) | [prompts/AGENTS.md](prompts/AGENTS.md) |
