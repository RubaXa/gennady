# Индекс

### 🎯 [ai-icl.md](./ai-icl.md)

Исследование **In-Context Learning (ICL)**: от **Few-Shot к Many-Shot**, **архитектура примеров**, проблемы **порядка и смещения**, **синтетический и негативный ICL**. Описывает техники **обучения моделей на примерах в контексте без изменения весов**.

### ⚡ [ai-priming.md](./ai-priming.md)

Руководство по **продвинутому праймингу и латентному управлению**: **семантическая, структурная, когнитивная, рекурсивная и заместительная** техники прайминга для настройки поведения LLM через **активацию специфических векторов и паттернов внимания**.

### 🏗️ [ai-protocol-structural-prompting-v1.md](./ai-protocol-structural-prompting-v1.md)

Спецификация протокола **структурного промптинга**: архитектура **XML-промптов**, модульность (**IncludeOnce**), логирование (**LDD**), **оркестрация и метрики качества**. Описывает проектирование промптов как **программных интерфейсов для управления вниманием трансформеров**.
