# TypeScript — конвенции

## Запрещённые конструкции (Node 22+, совместимость с декораторами)

| Запрещено                                                        | Вместо                               |
| ---------------------------------------------------------------- | ------------------------------------ |
| `enum`                                                           | `const` + `as const` или union type  |
| Модификаторы в конструкторе (`constructor(public x, private y)`) | Явное объявление полей в теле класса |
| `private` / `#private` field                                     | `protected _field` + префикс `_`     |
| `readonly` в параметре конструктора                              | `readonly` в объявлении поля         |

## Паттерн для «приватных» полей

```ts
class Foo {
  protected _value: string; // не private, не #value
  constructor(value: string) {
    this._value = value; // явная инициализация
  }
}
```

**Причина:** в JS нет полноценной рефлексии; `protected` + `_` позволяет декораторам и метапрограммированию работать с полями.
